# Misty Team - Website Portfolio & Services

## Cấu trúc dự án

### Modules chính:
- **services/** - Dịch vụ (thiết kế website, app, tool, cho thuê web)
- **projects/** - Showcase dự án đã làm
- **api-marketplace/** - API marketplace để bán API tích hợp
- **tools/** - Các công cụ có sẵn
- **contact/** - Form liên hệ
- **sub-domain/** - Portfolio cá nhân (subdomain riêng)

### Assets:
- **assets/** - Tài nguyên chung (CSS, JS, images, fonts)
- Mỗi module có assets riêng trong `[module]/assets/`

### Includes:
- **includes/** - Các file include chung (header, footer, navigation, functions)

### Admin:
- **admin/** - Admin panel để quản lý nội dung

### Database:
- **database/** - Schema và migration files
